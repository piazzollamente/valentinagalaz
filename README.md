# Sitio web Valentina Galaz

Versión estática corregida para GitHub Pages.

## Publicación en GitHub Pages

Subir el contenido de esta carpeta directamente a la raíz del repositorio:

```txt
index.html
assets/
favicon.ico
robots.txt
placeholder.svg
.nojekyll
README.md
```

No subir la carpeta contenedora completa.

## Visualización

URL de GitHub Pages:

```txt
https://piazzollamente.github.io/valentinagalaz/
```

## Nota técnica

Las rutas de CSS, JS e imágenes son relativas para funcionar dentro del subdirectorio `/valentinagalaz/`.
